# Statistics, Quantitative Analytics, and Predictive Analytics

A hands-on repo progressing from Statistics to Quantitative Analytics to Predictive Analytics, applied to energy trading data science.

## Skills Covered

1. **Statistics: Descriptive & Diagnostic Analytics**  
   Tools: Excel, Pandas, Statsmodels  
   Focus: EDA, summaries, hypothesis testing.

2. **Quantitative Analytics: Quant Modeling**  
   Tools: ARIMA (Statsmodels), NumPy, SciPy  
   Focus: Time-series, correlations, risk simulations (e.g., VaR).

3. **Predictive Analytics: Modeling & Forecasting**  
   Tools: Logistic Regression (scikit-learn), Prophet, TensorFlow (LSTM)  
   Focus: Future predictions, e.g., price forecasting.

## Project Pipeline
End-to-end:  
- **Stats**: EDA on energy data (Pandas/Statsmodels).  
- **Quant**: Regression for correlations (NumPy/SciPy).  
- **Predictive**: LSTM for 7-day forecasts (TensorFlow).  

## Datasets
- EIA.gov: Energy stats.  
- Yahoo Finance: Prices via `yfinance` (no API key needed).  

Run `pip install -r requirements.txt` to start. Contributions welcome!
